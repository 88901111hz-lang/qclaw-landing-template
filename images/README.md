# 图片资源

请将以下图片放入此文件夹：

| 文件名 | 说明 | 尺寸建议 |
|--------|------|----------|
| `logo.png` | Logo 图标 | 88x88px |
| `logo-text.png` | 文字 Logo | 高度 48-80px |
| `tencent-logo.png` | 品牌图标 | 20x20px |
| `qrcode-beta.png` | 二维码 | 160x160px |

## 图片要求

- **格式**: PNG（支持透明）或 WebP
- **背景**: 透明
- **Logo**: 圆角正方形，22px 圆角
